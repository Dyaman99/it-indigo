State Park Project Feedbacks Received

Feedback 1

They liked my website layout and colors. They also liked the logo I made using canva. There was not much to add or change. One thing they noticed was that from the logo and the site name in the upper right corner, only logo was clickable. So they adviced me to make the site title clickable as well, where it should be linked to the homepage. Another thing they told me was that on my homepage, the area under the events section looked kind of empty, so they suggested that I should add more events and extend the event section.

Feedback 2

My website was very well made to them, but they particularly liked how I set up the footer. I had the logo, contact, and the quick links menu on my footer, and I had styled them carefully. This person also noted that the way I layed out the events section to the side made it look a little empty. So I had to brainstorm ideas on how to restyle my homepage layout so that it looks nicer. What i did in the end was that I made my events section sticky as we scroll. This helped the clients see events at all times as they navigate through the page, but it also improved the design of the page. Another important feedback I received was the location link on the nav bar. It was missing an icon which would improve the user interface.
